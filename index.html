How it works

Three steps.
One seamless flow.

Sasula separates the request from the payment — so anyone can pay you, using any Mobile Money provider, with full context attached.

STEP 01
Create your request
Enter the amount, a clear reason, your name or business, and an optional expiry. Tap to try it now.
STEP 02
Share a code or QR
Get a code like KLA-4829 and a QR. Share via WhatsApp, SMS, print it, write it on a board — works everywhere.
STEP 03
Payer pays their way
The payer picks MTN, Airtel, or Stanbic and pays. Both get a receipt instantly. Track everything in Receive Payments.
